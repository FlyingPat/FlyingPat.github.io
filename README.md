# FlyingPat.github.io

Marketing and privacy policy pages for **World of Tripeaks**.

- **Marketing:** https://flyingpat.github.io/
- **Privacy Policy:** https://flyingpat.github.io/privacy

## Setup

1. Create a public GitHub repo named exactly `FlyingPat.github.io`
2. Push these files to `main`
3. GitHub Pages is enabled automatically for `*.github.io` repos — no settings change needed
4. Update `index.html` line with `[APP_ID]` once the app is live on the App Store

## Files

| File | Purpose |
|------|---------|
| `index.html` | Marketing landing page |
| `privacy.html` | Privacy policy (required by App Store) |
| `_config.yml` | Jekyll title |
